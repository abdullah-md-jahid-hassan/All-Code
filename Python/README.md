A modular, GUI-based shortcut application to speed up repetitive or common development tasks. Choose a category (Python, Django, PHP), view available commands, and execute them in one click.

---

## 🧠 Project Overview

The **Shortcut App** is a developer tool that helps automate environment-specific tasks by grouping them into categories. Each category (like Python or Django) provides a list of quick actions such as running servers, initializing projects, or clearing caches.

---

## 🖥️ Key Features

1. **Dynamic Shortcut Groups**: Load categories like Python, Django, and PHP from folder structure.
2. **GUI Interface**: Built using Tkinter for easy and intuitive navigation.
3. **Directory Management**: Shows current working directory with the ability to change it.
4. **Action Launcher**: Click to trigger scripts or commands specific to the selected group.
5. **Modular Design**: Easily extend by adding new folders and shortcut definitions.
